# Pokedex CI/CD - version portfolio

Base issue des ressources du TP4 "Développement web et dispositifs interactifs".

## Ce que montre le projet
- application web autour d'un Pokédex
- intégration avec API
- tests unitaires (Vitest)
- tests end-to-end (Playwright)
- lint avec ESLint
- build Vite
- CI GitHub Actions

## Installation
npm install
npm run dev

## Vérifications
npm run lint
npm test
npm run build

## Pour le CV
Projet Pokédex réalisé dans le cadre d'un TP de développement web, avec mise en place d'une chaîne de qualité incluant lint, tests et intégration continue.
